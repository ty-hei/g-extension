Smart Sidebar Assistant Privacy Policy

Effective Date: June 10, 2025

This policy is also available in Chinese (中文).

Welcome to Smart Sidebar Assistant (hereinafter referred to as "this extension"). We value your privacy and are committed to protecting your personal information.

Please read this Privacy Policy carefully and fully understand its terms before using this extension.

1. Information We Collect

Important Note: All information is stored locally in your browser or sent directly by you to a third-party AI service. We (the extension developers) do not have any servers, and we never collect, view, or store any of your personal data or API keys.

User Configuration Information

  • API Configurations: The API settings you add on the options page, including Configuration Name, API Key, API Type, API Endpoint URL (if applicable), and Model Name. This information is stored in chrome.storage.sync to be synced across your Chrome browsers where you are logged into the same Google account.
  • Active Configuration ID: An identifier for the currently active API configuration. This is also stored in chrome.storage.sync.

User-Generated Content and History

  • Conversation History: The complete record of your conversations with the AI. This is stored in your local browser via chrome.storage.local.
  • Archived Chats: Conversations or Q&A pairs that you manually archive. This is also stored in chrome.storage.local.
  • Prompt Templates: The prompt templates you create and save on the management page. This is stored in chrome.storage.local.

Information Processed for Functionality

  • Web Page Content: When you request to "Summarize Current Page" or "Extract Full Text," the extension reads the text content of the current page for processing.
  • Link Content: When you request to summarize a link, the extension temporarily opens the link in the background to extract its text content.
  • Selected Text or Images: When you use the context menu to analyze an image or select a piece of text on a web page, the corresponding image URL or text content is retrieved by the extension for subsequent AI analysis.

2. How We Use Your Information

We use your information solely to enable the features of the extension:

  • Connecting to AI Services: Your API key and configuration details are used to send requests directly from your browser to the third-party AI service you have chosen (e.g., Google Gemini or an OpenAI-compatible API).
  • Executing Your Instructions: The text you type, the content you select, and the web page content or image URLs you want to process are sent as part of a request to the AI service to get the desired response (e.g., a summary, translation, or analysis).
  • Maintaining Conversation Context: Your conversation history is used to build the context for requests sent to the AI, allowing it to understand previous interactions for a coherent dialogue.
  • Providing Personalized Features: Prompt templates help you quickly perform common tasks. Archiving and history features allow you to review important information.

3. Information Storage and Security

  • Local Storage: Your conversation history, archives, and prompt templates are stored on your local computer using the chrome.storage.local API. This means the data does not leave your device unless you initiate an action that sends it to an AI service.
  • Sync Storage: Your API configurations are stored using the chrome.storage.sync API. This data is synced across devices where you are logged into the same Google account and is protected by Google's security measures. The developers cannot access this information.
  • Security Measures: We follow industry-standard practices to protect your information. For example, the API key input field on the options page is a password field (type="password"). The extension contains no code that could send your data to the developers.

4. Information Sharing and Disclosure

We do not share or sell your personal information with any third party.

The core working model of this extension is to act as a "bridge" between you and a third-party AI service. When you use this extension, you acknowledge that:

  • The data you request to be processed (such as web page content or text you type) and your API key are sent directly from your browser to the third-party AI service provider you have configured (e.g., Google or another OpenAI-compatible service).
  • These third-party service providers have their own independent privacy policies and data-handling practices. We strongly recommend that you review their privacy policies before use. This extension is not responsible for the data processing practices of any third-party service.

To reiterate, none of your data ever passes through our servers.

5. Your Rights and Choices

You have complete control over your information:

  • Access and Modify: You can view, modify, or delete your API configurations at any time on the extension's "Options" page. You can also manage your templates on the "Manage Prompts" page.
  • Deletion:
    • You can permanently delete all conversation history using the "Clear All History" button in the sidebar.
    • You can delete individual archived chats or clear all archives on the "Archived Chats" page.
    • Uninstalling the extension will remove all data stored in chrome.storage.local (including history, archives, and templates).

6. Children's Privacy

This extension is not intended for use by children under the age of 13. We do not knowingly collect any personally identifiable information from children under 13.

7. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will post any changes on this page. We encourage you to review this Privacy Policy periodically to stay informed.
