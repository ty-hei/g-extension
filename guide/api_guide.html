API 密钥配置指南

为了使用智能侧边栏助手，您需要从 AI 服务提供商（如 Google 或 OpenAI）获取一个 API 密钥，并在插件中进行配置。本指南将引导您完成整个过程。

重要提示：您的 API 密钥非常敏感，请妥善保管，切勿泄露给他人。本插件会将密钥安全地存储在您的本地浏览器中，我们绝不会收集您的密钥。

获取 Google Gemini API 密钥

  1. 访问 Google AI for Developers 官方网站。
  2. 点击页面上的“Get API key in Google AI Studio”按钮，使用您的 Google 账户登录。
    [截图：显示 Google AI for Developers 页面和 "Get API key" 按钮]
  3. 登录后，您将进入 Google AI Studio。在左侧导航栏或主界面找到并点击 “Get API key” 选项。
  4. 在 API 密钥页面，点击 “Create API key in new project” 按钮来创建一个新的密钥。
    [截图：显示 Google AI Studio 中创建新 API 密钥的按钮]
  5. 密钥生成后，点击它旁边的复制按钮。现在您已经获得了 Gemini API 密钥。

获取 OpenAI 兼容 API 密钥

此方法适用于 OpenAI 官方 API 以及任何与 OpenAI API 格式兼容的第三方服务。

  1. 访问您选择的 AI 服务提供商网站（例如 OpenAI Platform）。
  2. 登录您的账户，并导航至 API Keys 管理页面。对于 OpenAI，您可以在登录后点击右上角头像，选择 “View API keys”。
    [截图：显示 OpenAI Platform 仪表盘和 API Keys 菜单位置]
  3. 点击 “Create new secret key” 按钮。您可以为这个密钥起一个名字（例如 "Browser-Sidebar-Key"）以便于识别。
  4. 创建后，请立即复制您的密钥。请注意，出于安全原因，您将无法再次完整地看到这个密钥。
    [截图：显示 OpenAI API 密钥创建和复制的界面]

在插件中配置密钥

  1. 在浏览器工具栏中，右键点击“智能侧边栏助手”的图标，然后选择“选项”。
    [截图：显示在浏览器工具栏右键点击插件图标并选择“选项”]
  2. 在打开的“API 配置管理”页面中，填写表单：
    • 配置名称: 给这个配置起一个易于识别的名字，如 “我的 Gemini 密钥”。
    • API 密钥: 粘贴您刚刚从服务商处复制的密钥。
    • API 类型: 根据您的密钥类型选择 "Google Gemini" 或 "OpenAI 兼容 API"。
    • API Endpoint URL: (仅当类型为 OpenAI 兼容时需要) 填写服务的 Endpoint 地址，例如 OpenAI 官方是 https://api.openai.com/v1/chat/completions。
    • 模型名称: 填写您想使用的具体模型 ID，例如 gemini-1.5-flash-latest 或 gpt-4o。
    [截图：显示插件的 API 配置页面，并高亮各个输入框]
  3. 点击“保存配置”。保存后，请确保您想使用的配置旁边显示为“(当前活动)”。如果不是，请点击该配置的“设为活动”按钮。

恭喜！配置完成后，您就可以打开侧边栏开始使用了。
